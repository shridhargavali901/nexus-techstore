## 🚀 About The Project
Nexus is a modern, high-performance e-commerce web application designed with a dark, cinematic SaaS-level UI. It provides a seamless shopping experience for premium tech gadgets, complete with a secure checkout flow and a robust, role-based Admin Command Center for inventory and order management.

## 💻 Tech Stack
**Frontend Architecture:**
* **React.js & Vite:** For lightning-fast rendering and an optimized development environment.
* **Tailwind CSS:** Fully responsive, modern styling with dark-mode aesthetics and glassmorphism effects.
* **Zustand:** Lightweight and efficient global state management (Cart & Authentication).
* **React Router v6:** Seamless single-page application (SPA) navigation.

**Backend Architecture:**
* **Node.js & Express.js:** Robust RESTful API architecture.
* **MongoDB & Mongoose:** Flexible NoSQL database for products, users, and orders.
* **JWT & Bcrypt:** Secure, token-based user authentication and password hashing.

## ✨ Key Features
* 🛡️ **Supreme Admin Dashboard:** Dedicated command center for the admin to add/delete products and clear fulfilled orders.
* 🔒 **Role-Based Access Control:** Secure routes ensuring regular users cannot access administrative functions.
* 🛒 **Dynamic Cart System:** Real-time cart updates, total calculations, and seamless checkout workflow.
* 🎨 **Cinematic UI/UX:** Premium design system featuring pulsing notifications, backdrop blurs, and luxury typography.

## ⚙️ Local Setup
1. Clone the repository.
2. Install dependencies for both client and server: `npm install`
3. Set up environment variables (`MONGO_URI`, `JWT_SECRET`).
4. Run the backend: `node server.js`
5. Run the frontend: `npm run dev`
